## 주요 테스트 시나리오

### 시나리오 1: 회원가입 ~ 로그인

| 단계 | 행위 | 기대 결과 | 결과 |
|---|---|---|---|
| 1 | 회원가입 API 호출 | 201 Created | |
| 2 | 동일 이메일 재가입 | 409 Conflict | |
| 3 | 로그인 API 호출 | 200 OK + 토큰 발급 | |

### 시나리오 2: 상품 등록 ~ 경매 ~ 낙찰

| 단계 | 행위 | 기대 결과 | 결과 |
|---|---|---|---|
| 1 | 상품 등록 | 201 Created | |
| 2 | 경매 등록 | 201 Created | |
| 3 | 입찰 | 200 OK | |
| 4 | 경매 종료/낙찰 | 낙찰 이벤트 발행 | |
| 5 | 주문 자동 생성 확인 | Order 서비스에 주문 생성됨 | |

### 시나리오 3: 결제 ~ 환불

| 단계 | 행위 | 기대 결과 | 결과 |
|---|---|---|---|
| 1 | 결제 요청 | 200 OK | |
| 2 | 예치금 차감 확인 | Member 서비스 잔액 감소 | |
| 3 | 환불 요청 | 200 OK | |
| 4 | 예치금 환급 확인 | Member 서비스 잔액 복구 | |
