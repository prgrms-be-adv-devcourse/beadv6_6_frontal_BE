# Biddy MSA 환경 구성

## 기술 스택

| 항목 | 버전 |
|---|---|
| Java | 21 (Eclipse Temurin) |
| Spring Boot | 3.4.1 |
| Spring Cloud | 2024.0.1 |
| Gradle | Wrapper 포함 |
| PostgreSQL | 16 |
| Apache Kafka | 3.7.0 (KRaft 모드) |
| Docker | 최신 버전 권장 |
| Docker Compose | v2 이상 |

## 프로젝트 구조

```
beadv6_6_frontal_BE/
├── discovery/      ← Eureka Server (서비스 등록/발견)
├── config/         ← Config Server (중앙 설정 관리)
├── apigateway/     ← API Gateway (라우팅/인증)
├── member/         ← 회원·인증·예치금
├── product/        ← 상품 등록·목록·검색
├── order/          ← 장바구니·주문
├── auction/        ← 경매·입찰·낙찰
├── payment/        ← 결제·환불·정산
├── init-db/        ← PostgreSQL 초기화 스크립트
├── k8s/            ← Kubernetes 배포 설정
├── docs/           ← 프로젝트 문서
├── .env            ← 환경변수 (git 제외)
├── .gitignore
├── docker-compose.yml
└── .dockerignore
```

## 포트 구성

| 서비스 | 포트 | 설명 |
|---|---|---|
| PostgreSQL | 5432 | 데이터베이스 |
| Kafka | 9092 | 메시지 브로커 |
| Discovery | 8761 | Eureka Server |
| Config | 8888 | Config Server |
| API Gateway | 8000 | 클라이언트 진입점 |
| Member | 8081 | 회원·인증·예치금 |
| Product | 8082 | 상품 등록·목록·검색 |
| Order | 8083 | 장바구니·주문 |
| Auction | 8084 | 경매·입찰·낙찰 |
| Payment | 8085 | 결제·환불·정산 |

## 서비스 간 통신

```
Client → API Gateway (8000) → 각 도메인 서비스
                                  ↕ (Eureka를 통한 서비스 발견)
                                  ↕ (Kafka를 통한 이벤트 통신)
```

## 주요 Dependencies (도메인 서비스 공통)

| 라이브러리 | 용도 |
|---|---|
| Spring Web | REST API |
| Spring Data JPA | DB 연동 |
| Spring Validation | 요청 검증 |
| Spring Actuator | 헬스체크/모니터링 |
| Spring Cloud Eureka Client | 서비스 등록/발견 |
| Spring Cloud Config Client | 중앙 설정 관리 |
| Spring Kafka | 이벤트 기반 통신 |
| PostgreSQL Driver | DB 드라이버 |
| Lombok | 보일러플레이트 코드 제거 |
| SpringDoc OpenAPI (Swagger) | API 문서 자동화 |

## 환경변수 (.env)

프로젝트 루트의 `.env` 파일에서 인프라 접속 정보를 관리합니다. (`.gitignore`에 포함)

| 변수 | 기본값 | 설명 |
|---|---|---|
| `POSTGRES_HOST` | `localhost` | PostgreSQL 호스트 |
| `POSTGRES_PORT` | `5432` | PostgreSQL 포트 |
| `POSTGRES_USER` | `biddy` | PostgreSQL 사용자 |
| `POSTGRES_PASSWORD` | `biddy1234` | PostgreSQL 비밀번호 |
| `POSTGRES_DB` | `biddy` | PostgreSQL 기본 데이터베이스 |
| `MEMBER_DB` | `biddy_member` | Member 서비스 DB |
| `PRODUCT_DB` | `biddy_product` | Product 서비스 DB |
| `ORDER_DB` | `biddy_order` | Order 서비스 DB |
| `AUCTION_DB` | `biddy_auction` | Auction 서비스 DB |
| `PAYMENT_DB` | `biddy_payment` | Payment 서비스 DB |
| `KAFKA_BOOTSTRAP_SERVERS` | `localhost:9092` | Kafka 브로커 주소 |

## 데이터베이스 구성 (서비스별 분리)

MSA 원칙에 따라 각 도메인 서비스가 독립된 데이터베이스를 사용합니다.

| 서비스 | 데이터베이스 |
|---|---|
| Member | `biddy_member` |
| Product | `biddy_product` |
| Order | `biddy_order` |
| Auction | `biddy_auction` |
| Payment | `biddy_payment` |

Docker Compose 최초 실행 시 `init-db/01-create-databases.sql` 스크립트가 자동으로 데이터베이스를 생성합니다.
